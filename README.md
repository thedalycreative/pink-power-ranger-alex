## 🛠 TECH STACK (The Command Centre)
From a web dev POV, this isn't just a website; it's a battle-hardened transmission platform.

### ⚛️ Frontend Architecture
- **Vite + React 18**: Because light-speed payments require light-speed bundling. We're using functional components and hooks (`useState`, `useEffect`, `useRef`) to manage the Morph Sequence state.
- **Tailwind CSS**: Atomic utility classes for maximum Ranger agility. No bloated CSS here, just pure, unadulterated performance.

### 🎨 Design System (Aesthetic Engineering)
- **Fluid Typography**: Implemented via `clamp(2rem, 12vw, 9rem)` to ensure the mission briefing is readable on everything from a smart-watch to a command centre projector.
- **Custom Shaders & FX**:
    - **Scanlines**: A CSS-based CRT overlay for that "classified transmission" feel.
    - **Grain Filter**: SVG turbulence filter applied as a pseudo-element for high-fidelity texture.
    - **Glitch Animations**: Keyframe-driven CSS animations for that "incoming signal" aesthetic.
- **Glassmorphism**: Backdrop blur effects (`backdrop-blur-md`) on all council statements for maximum clarity.

### 📱 Responsiveness
Battle-tested for mobile. We know Alex is on the go, so we've implemented:
- **Responsive Grids**: Collapsing from multi-column to single-column layouts.
- **Touch-Friendly Targets**: Large, glowing interactive elements designed for human (and potentially paw) interaction.
- **Zero-Overflow Policy**: Using `overflow-x-hidden` and `break-words` to ensure the transmission doesn't leak.
